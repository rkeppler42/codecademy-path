# 🗽 New York City Blog

> **EN:** Guided project from the [Codecademy Front-End Engineer](https://www.codecademy.com/learn/paths/front-end-engineer-career-path) career path — Semantic HTML module.
>
> **PT-BR:** Projeto guiado do path [Front-End Engineer](https://www.codecademy.com/learn/paths/front-end-engineer-career-path) da Codecademy — módulo de HTML semântico.

---

## 📋 EN

A static blog page about New York City, built to practice semantic HTML elements and embedded media.

**Concepts practiced:**

- Semantic layout elements (`<header>`, `<nav>`, `<main>`, `<footer>`)
- Content sectioning (`<section>`, `<article>`, `<aside>`)
- Media elements (`<figure>`, `<figcaption>`, `<video>`, `<audio>`, `<embed>`)
- Navigation with internal anchor links (`href="#id"`)
- Ordered lists (`<ol>`)
- `alt` attributes on images

---

## 📋 PT-BR

Página estática de blog sobre Nova York, criada para praticar elementos semânticos de HTML e mídia embutida.

**Conceitos praticados:**

- Elementos semânticos de layout (`<header>`, `<nav>`, `<main>`, `<footer>`)
- Seccionamento de conteúdo (`<section>`, `<article>`, `<aside>`)
- Elementos de mídia (`<figure>`, `<figcaption>`, `<video>`, `<audio>`, `<embed>`)
- Navegação com âncoras internas (`href="#id"`)
- Listas ordenadas (`<ol>`)
- Atributo `alt` em imagens
