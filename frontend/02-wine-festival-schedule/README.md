# 🍷 Wine Festival Schedule

> **EN:** Guided project from the [Codecademy Front-End Engineer](https://www.codecademy.com/learn/paths/front-end-engineer-career-path) career path — HTML tables module.
>
> **PT-BR:** Projeto guiado do path [Front-End Engineer](https://www.codecademy.com/learn/paths/front-end-engineer-career-path) da Codecademy — módulo de tabelas HTML.

---

## 📋 EN

A static event schedule page for a fictional wine festival, built to practice HTML table structure and organization.

**Concepts practiced:**

- Table structure (`<table>`, `<thead>`, `<tbody>`)
- Table rows and cells (`<tr>`, `<th>`, `<td>`)
- Column spanning with `colspan`
- CSS class targeting on table cells
- Google Fonts integration
- Fixed footer layout

---

## 📋 PT-BR

Página estática de agenda de eventos para um festival de vinhos fictício, criada para praticar estrutura e organização de tabelas HTML.

**Conceitos praticados:**

- Estrutura de tabela (`<table>`, `<thead>`, `<tbody>`)
- Linhas e células de tabela (`<tr>`, `<th>`, `<td>`)
- Mesclagem de colunas com `colspan`
- Seleção por classe CSS em células de tabela
- Integração com Google Fonts
- Layout de footer fixo
