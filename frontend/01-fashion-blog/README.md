## 📋 EN

A static blog page for a fictional fashion blogger, built to practice core HTML concepts: document structure, semantic elements, headings hierarchy, links, images, lists, and anchor navigation.

**Concepts practiced:**

- HTML document structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Heading hierarchy (`<h1>` through `<h4>`)
- Images with `alt` attributes
- Internal anchor links (`href="#id"`)
- External links with `target="_blank"`
- Unordered lists
- `<div>` with `id` for section targeting
- Inline text emphasis with `<strong>`

---

## 📋 PT-BR

Página estática de um blog de moda fictício, criada para praticar os conceitos centrais de HTML: estrutura de documento, elementos semânticos, hierarquia de headings, links, imagens, listas e navegação por âncoras.

**Conceitos praticados:**

- Estrutura de documento HTML (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Hierarquia de headings (`<h1>` a `<h4>`)
- Imagens com atributo `alt`
- Links de âncora internos (`href="#id"`)
- Links externos com `target="_blank"`
- Listas não ordenadas
- `<div>` com `id` para navegação por seção
- Ênfase de texto inline com `<strong>`
